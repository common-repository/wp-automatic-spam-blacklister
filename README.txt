The WP Spam Blacklister is a simple spam-prevention plugin that adds the IP address of a marked spam post on Wordpress' comments blacklist.

== Description ==
If you like total control of the comments on your Wordpress blog without having any extra overheads (like you get with 3rd party spam prevention software) then this is a great tool for you.

= Features =
*    Automatically placed IP address on comments blacklist if administrator marks comment as spam
*    Will remove IP address from comments blacklist if spam item is marked as 'not spam'
*    When activated, will review all spam comments and place the spammers IP address on the comments blacklist.

= Support =
[All support is provided through the plugin developer's website](http://www.blueliquiddesigns.com.au/index.php/wp-automatic-spam-blacklister/).

== Installation ==

1. Install and activate plugin on WordPress
2. Go to Settings -> Discussion in the admin area and review the newly-added IP addresses on in the Comments Blacklist box.
3. When you mark comments as spam a user's IP address will automatically be placed on the comments blacklist.
